# WL Building Solutions — Business Website & AI Chatbot

A full-stack web application built for a real Melbourne renovation business.

## Live Site
🌐 [wlbuildingsolutions.com](http://wlbuildingsolutions.com)

## What I Built
- 4-page professional website (Home, Services, Gallery, Contact)
- AI chatbot powered by Claude API that answers customer FAQs and collects leads
- Contact form with backend data capture saved to CSV
- Fully mobile responsive design
- Cloud deployed with custom domain

## Technologies Used
- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript
- **AI:** Claude API (Anthropic)
- **Deployment:** Railway
- **DNS:** Cloudflare
- **Version Control:** Git, GitHub

## Features
- AI chatbot handles customer enquiries 24/7
- Lead collection via chat and contact form
- Mobile friendly across all screen sizes
- Production deployed on cloud infrastructure
